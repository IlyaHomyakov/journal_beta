Это веб-клиент приложения для поиска расписаний учебных групп Белорусского Государственного Медицинского Университета.

Для того, чтобы найти необходимое Вам расписание достаточно в поле поиска ввести номер вашей группы и нажать Enter или на кнопку поиска. Если Ваша группа находится в базе, будет выведено расписание для введенной группы на две недели вперед.

Приложение находится в разработке (!)
А это значит, что, вероятно, не все группы добавлены в базу. Если Вы студент БГМУ и хотите помочь приложению (добавить расписание / сообщить об ошибке / другая причина), то по этой ссылке можно связаться с разработчиками.
